📲 1단계 — 앱 혹은 즐겨찾기 30초

홈 화면 아이콘으로 한 번에 열 수 있어요. 오프라인에서도 마지막 시간표가 보입니다.
앱 설치가 부담되면 그냥 브라우저 즐겨찾기(북마크)에 추가해서 봐도 똑같이 잘 보여요.

iPhone / iPad
  1. Safari로 위의 [시간표 바로 열기]를 누르세요 (크롬 ❌ → 사파리여야 해요)
  2. 상단/하단의 공유 ⬆️ 버튼
  3. 더보기 누르기
  4. 홈 화면에 추가 → 추가
안드로이드
  1. Chrome으로 시간표 열기
  2. 우상단 ⋮ 메뉴 → 홈 화면에 추가 (또는 "앱 설치")
Windows / Mac (노트북)
  1. Chrome: 우상단 ⋮ 점 세 개 → 전송, 저장, 공유 → 페이지를 앱으로 설치
  2. Edge: 우상단 ⋯ 점 세 개 → 기타 도구 → 앱 → 이 사이트를 앱으로 설치
  3. 설치 후 작업 표시줄에 고정하면 더 편해요 — 일반 프로그램처럼 실행됩니다
설치 없이 즐겨찾기로만 보기
  1. 브라우저로 시간표를 연 뒤
  2. 아이폰 Safari: 공유 ⬆️ → 즐겨찾기에 추가 / PC: Ctrl+D (Mac은 ⌘+D)
  3. 볼 때마다 즐겨찾기에서 열면 끝 — 기능은 앱과 동일해요

📱 2단계 — 홈 화면 위젯 선택 · 2분

앱을 열지 않아도 홈 화면에서 이번 주 시간표가 바로 보여요. (iPhone/iPad 전용)

이 앱이 맞아요 → Scriptable (무료)
개발자 Simon B. Støvring · 검은 배경에 주황색 곡선 아이콘
App Store에서 열기 ↗

⚠️ 이름이 비슷한 다른 앱을 받지 마세요 — 위 아이콘과 개발자명을 꼭 확인!

  1. App Store에서 Scriptable (무료) 설치 — 위 아이콘이 맞는지 확인
  2. 아래 [코드 복사] 버튼을 누르세요
  3. Scriptable 열기 → 우상단 ＋ → 빈 화면에 붙여넣기
  4. 좌상단 이름을 시간표로 바꾸고 완료
  5. 홈 화면 빈 곳 길게 누르기 → 좌상단 ＋ → Scriptable 검색 → 크기 선택 → 추가
    아이폰은 가로로 긴 위젯, 아이패드는 초대형 추천!
  6. 추가된 위젯 길게 누르기 → 위젯 편집 → Script: 시간표 선택
코드 불러오는 중…
  • 자정이 지나면 위젯이 알아서 오늘 날짜·이번 주로 넘어가요 — 날짜를 따로 맞출 필요가 전혀 없습니다. 새 주가 시작되면 다음 주 시간표로 자동 전환!
  • 위젯은 30분~1시간 간격으로 자동 갱신돼요. 위젯을 탭하면 전체 시간표가 열립니다.
  • 위젯 기능이 업그레이드되면 자동으로 반영됩니다 — 재설치 필요 없음.
🔮 다음 주, 다다음 주도 위젯으로 보기 (스택으로 넘기기)
위젯을 하나 더 추가하고 위젯 편집 → Parameter 칸에 1을 입력하면 그 위젯은 다음 주를 보여줘요 (2 = 2주 뒤, 3 = 3주 뒤…).
그리고 이번주·다음주 위젯을 같은 자리에 겹쳐 쌓으면(스마트 스택), 위젯을 위아래로 쓸어넘겨서 이번 주 → 다음 주 → 다다음 주를 휙휙 넘겨볼 수 있어요!

🧭 사용법

주간이번 주 전체 (기본 화면). ◀ ▶로 한 주씩 이동

월간이번 달 달력 — 칩 위치가 위면 오전, 아래면 오후 수업. 날짜를 누르면 그 주로 점프. ◀ ▶로 한 달씩
칸 터치과목명·교수명·과명·초안자/검안자가 크게 펼쳐져요
색파랑 글씨 = 초안자/검안자·점심 그룹 · 빨강 = 시험/평가 · 오늘 요일엔 파란 테두리
📱 화면이 작아 글씨가 작게 보여도 걱정 마세요! 칸을 한 번 터치하면 그 수업의 과목명·교수명·과명·초안자/검안자가 큼직하게 펼쳐집니다.

💬 FAQ 및 주의사항

  • 과거 시간표가 보여요 → 앱을 완전히 닫았다가 한 번만 다시 열어보세요.
  • 아이폰에서 글자가 너무 작아요 → 확대도 가능하고, 핸드폰을 가로모드로 보면 커져요! 아이패드 가로로 보기 좋은 사이즈이고 PC 화면에선 공간이 남을 거예요.. 그래도 불편하다면 문의주세용
  • 위젯에서는 오늘/월간 보기는 안 되나요? → 위젯은 빠르게 주간 시간표와 공강을 확인하는 용도예요. 대신 다음 주는 볼 수 있어요 — 위 2단계의 "🔮 다음 주도 위젯으로 보기"를 참고!
  • 위젯이 비어 있어요 → 인터넷 연결을 확인하고, Scriptable 앱에서 "시간표"를 한 번 실행해보세요.
  • iPhone에서 "홈 화면에 추가"가 없어요 → 크롬이 아닌 Safari로 열어야 해요.
  • 기타 문의사항/개선사항 → 정말 뭐든지 좋아요! 전부 다 GY(강윤)에게 편하게 카톡주세용
  • !주의1! 링크를 유출하지 말아주세요.. 일정 확인용으로 개발했답니다
  • !주의2! 그룹 공지와 공식 시간표를 보는 게 더 빠르게 변경사항을 확인하는 방법입니다.. GY가 가독성 좋게 변경 및 적용하는 데 시간이 걸리므로 양해바랍니다
