의학과 2학년 시간표

시간표가 바뀌면 1분 안에 자동 반영되는 실시간 시간표예요.
한 번만 설치하면 끝!

▶  시간표 바로 열기 https://pureart-art.github.io/Timetable26-1/

📲 1단계 — 앱처럼 설치하기 30초

홈 화면 아이콘으로 한 번에 열 수 있어요. 오프라인에서도 마지막 시간표가 보입니다.

iPhone / iPad
  1. Safari로 위의 [시간표 바로 열기]를 누르세요 (크롬 ❌ → 사파리여야 해요)
  2. 하단 가운데 공유 ⬆️ 버튼
  3. 아래로 내려서 홈 화면에 추가 → 추가
안드로이드
  1. Chrome으로 시간표 열기
  2. 우상단 ⋮ 메뉴 → 홈 화면에 추가 (또는 "앱 설치")
Windows / Mac (노트북)
  1. Chrome 또는 Edge로 시간표 열기
  2. 주소창 오른쪽 끝의 설치 ⊕ 아이콘 클릭 → 설치
  3. 이후 일반 프로그램처럼 실행됩니다

📱 2단계 — 홈 화면 위젯 선택 · 2분

앱을 열지 않아도 홈 화면에서 이번 주 시간표가 바로 보여요. (iPhone/iPad 전용)

  1. App Store에서 Scriptable (무료) 설치
  2. 아래 [코드 복사] 버튼을 누르세요
  3. Scriptable 열기 → 우상단 ＋ → 빈 화면에 붙여넣기
  4. 좌상단 이름을 시간표로 바꾸고 완료
  5. 홈 화면 빈 곳 길게 누르기 → 좌상단 ＋ → Scriptable 검색 → 크기 선택 → 추가
    아이폰은 가로로 긴 위젯, 아이패드는 초대형 추천!
  6. 추가된 위젯 길게 누르기 → 위젯 편집 → Script: 시간표 선택
코드 불러오는 중…
  • 위젯은 30분~1시간 간격으로 자동 갱신돼요. 위젯을 탭하면 전체 시간표가 열립니다.
  • 위젯 기능이 업그레이드되면 자동으로 반영됩니다 — 재설치 필요 없음.

🧭 사용법

오늘오늘 하루 시간표. ◀ ▶로 하루씩 이동
주간이번 주 전체 (기본 화면). ◀ ▶로 한 주씩 이동
월간이번 달 달력 — 칩 위치가 위면 오전, 아래면 오후 수업. 날짜를 누르면 그 날로 점프. ◀ ▶로 한 달씩
칸 터치과목·교수·초안자/검안자 전체 텍스트가 크게 떠요
색파랑 글씨 = 초안자/검안자·점심 그룹 · 빨강 = 시험/평가 · 오늘 요일엔 파란 테두리

🛟 잘 안 될 때

  • 화면이 옛날 시간표예요 → 앱을 완전히 닫았다가 두 번 정도 다시 열어보세요.
  • 위젯이 비어 있어요 → 인터넷 연결을 확인하고, Scriptable 앱에서 "시간표"를 한 번 실행해보세요.
  • iPhone에서 "홈 화면에 추가"가 없어요 → 크롬이 아닌 Safari로 열어야 해요.

시간표 원본이 수정되면 모두의 화면에 자동 반영됩니다 🔄
