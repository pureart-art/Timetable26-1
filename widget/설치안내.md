# 📱 시간표 위젯 설치 안내 (iPhone / iPad)

홈 화면에서 앱을 열지 않고도 시간표가 보이는 위젯입니다.
한 번만 설치하면 시간표가 바뀌어도, 위젯이 개선되어도 **자동으로 업데이트**됩니다.

## 설치 (2분, 한 번만)

1. App Store에서 **Scriptable** (무료) 설치
2. Scriptable 열기 → 우상단 **+** → 빈 화면에 **아래 코드 전체를 붙여넣기**
   - 코드: https://pureart-art.github.io/Timetable26-1/widget/loader.js 열어서 전체 복사
3. 좌상단 이름을 **시간표**로 바꾸고 완료(Done)
4. 홈 화면 빈 곳을 **길게 누르기** → 좌상단 **+** → **Scriptable** 검색 → 원하는 크기 선택 → 추가
5. 추가된 위젯을 **길게 누르기 → 위젯 편집** → **Script: 시간표** 선택 (Parameter는 비워두면 됨)

모든 크기에서 **이번 주 주간 시간표**가 나옵니다. 크기가 클수록 글자가 잘 보입니다:
- iPhone: **중형(가로로 긴 위젯)** 이상 추천
- iPad: **초대형(extraLarge)** 추천 — 표가 가장 시원하게 보임
- 오늘 요일에 파란 테두리 표시
- 위젯을 **탭하면** 전체 시간표 앱이 열립니다: https://pureart-art.github.io/Timetable26-1/

## 참고

- 위젯 갱신 주기는 iOS가 관리합니다(보통 15분~1시간). 실시간 확인이 필요하면 위젯을 탭해서 앱으로.
- 오프라인이면 마지막으로 받아온 시간표를 보여줍니다.
- 위젯이 "시간표를 불러오지 못했어요"라면: 인터넷 연결 확인 → Scriptable 앱에서 스크립트를 한 번 실행해보기.
